GIT-HUB





Git es un sistema de control de cambios y en combinación con GIHUB, se tiene una plataforma de desarrollo colaborativo.

Github fue escrito en el lenguaje ruby on Rails, también conocido como RoR o Rails, un framework de código abierto.

Git-hub anteriormente se llamaba Logical Awesome LLC. Esta grandiosa aplicación fue comprada por Microsoft En 2018 (ver video)

el cual permitió publicar proyectos donde es posible acceder a códigos fuentes.
