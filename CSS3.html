CSS3





CSS son las siglas de “Hojas de estilo en cascada” y que sirve para dar estilo a nuestra estructura creada en HTML. CSS es la última versión hasta la fecha y presenta como principales características mayor control sobre el estilo de los elementos de nuestra página web y mayor número de efectos visuales. Las ventajas de usar CSS3 son: Código más simple, mayores opciones de gráficas.
