LENGUAJE HTML







Es un lenguaje que se utiliza para la creación de páginas en la WWW.

Se compone de una serie de comandos que son interpretados por el programa que se usa para navegar por internet. Pero este navegador puede interpretar adecuadamente cada instrucción, sin embargo, hay casos donde los navegadores no interpretan el código de ciertas páginas web, esto se aprecia cuando una página web funciona en determinado navegador y no lo hace en otro.
